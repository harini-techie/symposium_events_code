Reverse Coding — Rules & Regulations

Code from a different perspective.

Participation & Task

  • Open to all UG & PG Students.
  • Individual person.
  • Participants will be given outputs, constraints, and logic descriptions.
  • They must write the code that produces the correct output.

Levels of Competition

  1. Level 1
    • 5 reverse coding questions.
    • Time limit: 15 minutes.
    • Only those who solve all questions correctly will qualify for the next level.
  2. Level 2
    • Qualified coders will face 3 new questions.
    • Time limit: 10 minutes.
    • Only successful solvers from this level will be promoted to the final level.
  3. Level 3
    • Finalists will be presented with 3 challenging reverse coding questions.
    • Time limit: 15 minutes.

Programming Language

  • Allowed languages: C, C++, Java, Python.
  • No restriction on language choice, but only one language must be used throughout the event.
